Morph3D Screen Saver v4.2

Morph3D Screen Saver - это скринсейвер для Windows.
При запуске заставки на экране изображаются различные трехмерные вращающиеся объекты. Всего их восемнадцать, в их список входят сферы, додекаэдр, икосаэдр, тор, спираль, куб, тетраэдр и другие. Все объекты состоят из светящихся точек. Один объект периодически превращается в другой (эффект "morphing"). Точки оставляют за собой след, который стирается одним из трех эффектов, самый красивый из которых-огонь.

Параметры настройки :
  • "Показывать FPS" - отображение числа кадров в секунду (при включенном эффекте следа - 40 "FPS")
  • "Перемешать точки". Если флажок установлен, то точки при превращении объектов движутся "хаотично". Иначе точки будут перемещаться небольшими группами
  • "Выход при перемещении мышки". Если флажок установлен, то при перемещении указателя мыши работа скинсейвера прекратится
  • "Перемещение в 3D-пространстве". При установке опции объект перемещается не только влево-вправо, вверх-вниз, но и вперед-назад. Если не нравится, можно отключить
  • "Упрощенная прорисовка точек" - только для медленных машин. Точки не круглые, а квадратные, из-за этого увеличивается FPS
  • "Градации серого" - точки не имеют цвета
  • "След"-точки оставляют за собой след, который стирается одним из следующих методов : "простой", "растворение" и "пламя". Вы также можете настроить его длину
  • Скринсейвер имеет защиту паролем
Системные требования :
  • Windows 95/98/NT/ME/2000/XP
  • Pentium 75 МГц или выше
